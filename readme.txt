== Description ==
Forces Internet Explorer 9 and higher to render the page as version 9.
This is done by adding the X-UA-Compatible meta tag.

== Installation ==
1. Upload `i9-mode.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
